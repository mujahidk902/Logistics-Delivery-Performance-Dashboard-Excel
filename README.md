# Logistics & Delivery Performance Dashboard

## Project Overview

This project analyzes 5,000+ e-commerce logistics orders using Microsoft Excel.

The dashboard provides insights into:

- Delivery Performance
- Cancellation Trends
- Product Category Performance
- Logistics Provider Analysis
- Transit Cost Analysis
- Total Logistics Cost

---

## Tools Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- KPI Cards
- Slicers

---

## Data Cleaning & Transformation

- Corrected data types
- Created Delivery in Days metric
- Created Delivery Performance metric
- Created Total Logistics Cost metric
- Replaced missing Logistics Provider values using Mode
- Removed invalid Transit Cost records
- Handled missing values using route-level business logic

---

## KPIs

- Total Orders
- Delivered Orders
- Cancelled Orders
- Cancellation Rate
- Average Transit Cost
- Total Logistics Cost

---

## Key Business Insights

- Mobile products generated the highest order volume.
- Mumbai received the highest shipment volume.
- TV product categories showed the highest cancellation rate.
- Laptop products generated the highest logistics costs.
- Nearly half of all orders were delivered before the promised date.
